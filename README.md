# Library Management System

## Setup Instructions

### 1. Install Dependencies


```bash
npm init -y
npm install express pg cors body-parser
npm install
```

### 2. Run the Application

```bash
node server.js
```

## Tech Stack
- Express.js - Backend server
- PostgreSQL - Database
- CORS - Cross-origin requests
- Body-parser - Request parsing
